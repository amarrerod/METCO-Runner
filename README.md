Overview
--------

Python module to run experiments in [METCO](https://github.com/PAL-ULL/software-metco)

Installation / Usage
--------------------

To install use pip:

    $ pip install metrunner


Or clone the repo:

    $ git clone https://github.com/marreA/metrunner.git
    $ python setup.py install
